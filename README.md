## How to run it on Ubuntu
`sudo apt update && sudo apt install cargo`
<br>

Create a project: <br>
`cargo new hackUPCgame` <br>
`cd hackUPCgame/` <br>

Then, run `./ubuntu_runner_rs --run src/main.rs` from project root.
